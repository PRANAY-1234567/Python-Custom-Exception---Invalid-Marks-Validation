# Python Custom Exception - Invalid Marks Validation

## 📌 Overview

This project demonstrates how to create and use a **custom exception** in Python for validating student marks.

The program accepts marks from the user and checks whether they are within the valid range of **0 to 100**. If the entered marks are outside this range, a custom `InvalidMarksException` is raised.

This project is useful for learning:

* Custom Exceptions
* Input Validation
* Exception Handling
* Object-Oriented Programming in Python

---

## 🚀 Features

* Accepts marks from the user
* Validates marks between 0 and 100
* Uses a custom exception class
* Displays meaningful error messages
* Beginner-friendly implementation

## 🔍 How It Works

1. User enters marks.
2. `check_marks()` verifies the range.
3. If marks are invalid, `InvalidMarksException` is raised.
4. The exception is caught and displayed.
